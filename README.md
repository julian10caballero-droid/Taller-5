# Taller-5
# Juego del Ahorcado en Consola 🪢

El proyecto consiste en el desarrollo de un juego interactivo de Ahorcado en Java, en el cual el usuario debe adivinar una palabra seleccionada aleatoriamente dentro de una categoría específica.
Las palabras y sus pistas se cargan desde un archivo externo en formato CSV, lo que permite que el contenido del juego sea dinámico y fácilmente modificable sin cambiar el código. 💻

# ⭐ Características principales

# 🎯 Mecánica del juego

- El usuario debe adivinar una palabra oculta letra por letra
- Cada acierto revela la posición correcta de la letra
- Cada error reduce el número de intentos disponibles

# ⏳ Control de intentos

- Número limitado de oportunidades
- Finaliza cuando el jugador adivina la palabra o pierde
- Validación de letras repetidas

# 🖥️ Visualización en consola

- Representación de la palabra con guiones y letras descubiertas
- Mensajes de acierto o error
- Uso de ASCII art representativo
